Если объявление D имени x принадлежит блоку B, то областью видимости
D является весь блок B, за исключением любого блока B', вложенного в
блок B на произвольной глубине, в котором повторно объявлено имя x.
Имя x повторно объявлено в блоке B' если этому блоку принадлежит некое
иное объявление D' того же имени x. То есть x во вложенном блоке
"перекрывает" собой объявление выше, и изменение локального x не влечёт
за собой изменения всех x, объявленных выше.

char x = '1'; // Объявление D имени x.
{
    char x = '2'; // Объявление D' имени x, создание нового объекта
                  // в памяти.
                  // Область видимости глобального x заканчивается
                  // после объявления локального x.
    ++x; // Использование наиболее близкого по уровню вложенности
         // объявления имени x.
}
++x; // Обращение к глобальному объявлению D имени x.
