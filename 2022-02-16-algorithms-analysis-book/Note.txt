Алгоритм - компьютерная программа (последовательность вычислительных шагов),
которая служит для преобразования входных данных в выходные.

Задача сортировки формально определяется как:
Вход: последовательность из n чисел (a0, a1, ..., an).
Выход: переупорядоченная последовательность (a'0, a'1, ..., a'n), такая, что
  a'0 <= a'1 <= ... <= a'n.

Входные данные также называются экземпляром (instance).

Алгоритм называется корректным, если для любых входных данных он даёт правильные
выходные данные. Если алгоритм некорректный, он может выдать неверные данные или
вовсе никогда не завершить свою работу.

В целом, сложность алгоритма с ростом размера входных данных влияет на скорость
его работы намного существенее, чем мощность вычислительной машины.
