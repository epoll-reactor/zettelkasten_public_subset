Сортировка файлов: ls -l
Сортировка файлов в обратном порядке: ls -lr
Сортировка файлов по размеру в порядке возрастания: ls -lS
Сортировка файлов по размеру в порядке убывания: ls -lSr
Сортировка файлов по времени их модификации: ls -lt
Сортировка файлов по расширению: ls -lX

Сортировка по строкам: sort file.txt
Сортировка по строкам в порядке убывания: sort -r file.txt
Сортировка чисел: sort -n file.txt
Сортировка по номерам версий:
  echo -e "1.0.0.1 \n 6.2.1.0 \n4.0.0.2" > text.txt
  sort --version-sort --field-separator=. text.txt

Поиск с расширением и сортировка: find . -iname "*.md" | sort -r
