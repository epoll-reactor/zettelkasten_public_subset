SWIFT-код - метод идентификации банков.

Код BIC представляет собой код вида BBBB CC LL (bbb), где
BBBB - код финансовой организации
CC   - код страны
LL   - код местонахождения
bbb  - код филиала(необязательно), если у организации его нет - значение равно XXX

Примеры:
  UBSWUS33XXX - SWIFT BIC
  UBSWUS33CHI - UBS AG Chicago branch (SWIFT BIC)
  VUNIRU31XXX - Волжский Универсальный Банк (NON-SWIFT BIC)
