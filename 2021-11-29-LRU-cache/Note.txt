LRU(Last Recently Used) кэш - способ хранения данных, при котором есть 3 случая:
1) Если кэшируемый элемент уже находится в кэше, тогда он сдвигается в начало.
2) Если кэш-линия уже заполнена, тогда удаляется последний (наиболее давно используемый)
   элемент.
3) Если элемента нет в кэше, он помещается в его начало.

Пример:
  Для подачи в кэш чисел 1, 2, 5, 1, 4, 5, кэш принимает состояния:
    [1]           1: промах, помещаем в начало
    [2, 1]        2: промах, помещаем в начало
    [5, 2, 1]     5: промах, помещаем в начало
    [1, 5, 2]     1: попадание, удаляем запрашиваемый элемент и помещаем в начало
    [4, 1, 5, 2]  4: промах, помещаем в начало
    [5, 4, 1, 2]  5: попадание, удаляем запрашиваемый элемент и помещаем в начало

  Операции получения чисел 5, 4, 1:
    [5, 4, 1, 2]  5: переместить пятёрку в начало -> [5, 4, 1, 2]
    [5, 4, 1, 2]  4: переместить четвёрку в начало, остальное сдвинуть -> [4, 5, 1, 2]
    [4, 5, 1, 2]  1: переместить единицу в начало, остальное сдвинуть -> [1, 4, 5, 2]
