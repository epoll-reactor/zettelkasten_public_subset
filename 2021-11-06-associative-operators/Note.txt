Некоторые операторы имеют свойство ассоциативности. Основные арифме-
тические операторы левоассоциативны (+, -, *, /). Левоассоциативность
- когда операнд со знаками "+" с обеих сторон используется левым
оператором.

Например, выражение 1 + 2 + 3 эквивалентно (1 + 2) + 3. Двойка склады-
вается с единицей, и результат этой операции складывается с стоящей
с правой стороны тройкой.

Некоторые операторы, такие как присваивание и возведение в степень -
правоассоциативны.

Например, для оператора присваивания в языке С можно создать грамма-
тику:
   right -> letter = right | letter
  letter -> a | b | ... | z

Деревья для лево- и правоассоциативных операторов:

            list                  right
          /  |  \               /   |   \
       list  - digit         letter =  right
      / | \     |              |      /  |  \
   list - digit 2              a     /   =   \
    |      |                      letter    right
  digit    5                        |         | 
    |                               b       letter
    9                                         |
                                              c
Дерево для арифметического выражения растёт влево, для цепочки
присваиваний - вправо.

Грамматику, учитывающую приоритеты операциий можно описать.
  factor -> digit | ( expr )

    term -> term * factor
          | term / factor
          | factor

    expr -> expr + term
          | expr - term
          | term

Таким образом, expr рассматривается как список элементов term,
разделённых операторами сложения или вычитания.
Итоговую грамматику можно записать как:
    expr -> expr +   term | expr -   term |   term
    term -> term * factor | term / factor | factor
  factor -> digit | ( expr )

factor - выражение, которое не может быть разорвано никаким операто-
ром. Factor может представлять собой выражение с * или /, выражение
в скобках, или отдельный операнд, который очевидно и так неделим.

term - выражение, которое может быть разорвано другим выражением с
более высоким приоритетом.
