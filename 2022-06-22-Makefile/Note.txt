Отключить вывод команды (не вывода):
	echo "OK"  # Выводит
	@echo "OK" # Не выводит

Проверка ошибок компиляции:
	if ! $(CC) main.c 2> /dev/null; then \
		echo "Catched compile error"; \
	else \
		echo "Expected compile error"; \
	fi

.PHONY-таргеты:
	Нужны затем, чтобы не создавать целевой таргет (позволяет
	внедрить какие-то оптимизации и избежать конфликт имён с реальными
	целями).

	Пример
	make_something:
		@command...
	.PHONY : make_something
